# Mission Esthetics

Private, detail-focused waxing and esthetics services in Rancho Peñasquitos.

## About

Mission Esthetics is a luxury esthetics studio offering personalized waxing services including Brazilian, bikini, full-body, brow, and facial waxing. All services are provided in a calm, private, judgment-free environment by Alexandra.

**Location:** Rancho Peñasquitos, California

**Services:** Brazilian Waxing, Brow Waxing, Full-Body Waxing, Bikini Waxing, and more

**Booking:** Available through GlossGenius — [Schedule a Session](https://missionesthetics.glossgenius.com/services)

## Development

This is a [Next.js](https://nextjs.org) project with a luxury editorial design system.

### Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Project Structure

- `app/` — Next.js 16 App Router pages and layout
- `components/` — React components (nav, hero, footer, sections)
- `app/globals.css` — Global styles and design tokens (dark/light mode support)
- `public/images/` — Image assets including the Mission Esthetics logo

### Design System

- **Typography:** Cormorant Garamond (serif), Inter (sans-serif)
- **Colors:** Dark luxury palette with blush, mauve, cream accents
- **Branding:** Rose bloom logo, consistent across all touchpoints
- **Responsive:** Mobile-first, fully responsive design for all screen sizes

### Build & Deploy

```bash
npm run build
npm run start
```
